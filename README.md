# Hackday Indra

Proyecto base en Python usando una estructura profesional con `src/ layout`, preparado para trabajo en equipo.

---

## Requisitos

- Python 3.10 o superior
- Git

Verifica tu versión de Python:
```bash
python --version
```

## Instalación

### 1. Crear y activar entorno virtual

**Windows:**
```bash
python -m venv .venv
.venv\Scripts\activate
```

**Linux/Mac:**
```bash
python -m venv .venv
source .venv/bin/activate
```

### 2. Instalar dependencias
```bash
pip install -r requirements.txt
```

### 3. Instalar el proyecto en modo desarrollo
```bash
pip install -e .
```

## Uso

Ejecutar el proyecto:
```bash
python -m hackday.main
```
